# **App Name**: VideoKoleks

## Core Features:

- Video Saving via Share: Saves videos from Instagram Reels, YouTube Shorts, and TikTok via the share button, capturing metadata.
- Intelligent Categorization: Automatically categorizes saved videos or lets the user assign videos to custom categories upon saving.
- Home Screen Video Grid: Displays saved videos in a customizable grid layout with thumbnail previews, platform badges, and category tags.
- Detail and Viewing Screen: Offers a full-screen video player with standard controls and additional video metadata.
- Category Management: Allows users to manage categories.
- Search Functionality: Enables searching for videos using keywords in titles, categories, or notes.
- Favorites and Profile Screens: Implements favorite screen and a profile screen to show their video counts and category stats.

## Style Guidelines:

- Primary color: Vivid purple (#8B5CF6) to evoke a youthful, energetic feel.
- Background color: Dark backdrop (#0F0F0F) for immersive video viewing.
- Accent color: Turquoise (#06B6D4) reminiscent of TikTok.
- Headline font: 'Inter' sans-serif bold for impactful titles and headers.
- Body font: 'Inter' sans-serif regular for a clean and readable interface.
- Simple, recognizable icons for navigation and actions.
- Fluid swipe transitions to change between videos on the detail page.
